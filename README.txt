HOW TO OPEN THE WEBSITE
-----------------------
Just double-click index.html in this folder.
It opens in any browser. No internet needed. Works offline.


HOW TO ADD KANIKA'S PHOTOS
---------------------------
1. Copy your photo files into THIS folder
   (same place as index.html)
   Name them anything: kanika1.jpg, birthday.jpg, etc.

2. Open script.js in any text editor (Notepad, VS Code, etc.)

3. Find the PHOTOS section near the top (around line 20):
   const PHOTOS = [
     // { src: 'kanika1.jpg', caption: 'My love', date: 'Jan 2024', wide: false },

4. Add your photos by removing the // from the example lines
   and filling in your filenames and captions:
   const PHOTOS = [
     { src: 'kanika1.jpg', caption: 'My favourite girl', date: 'Jan 2024', wide: true },
     { src: 'kanika2.jpg', caption: 'Always smiling', date: 'Apr 2024', wide: false },
   ];

5. Set wide: true for the FIRST/HERO photo (it spans full width).
   Set wide: false for all other photos.

6. Save script.js and refresh the browser.

7. For the POLAROID WALL — open index.html, find each:
   <div class="placeholder-polaroid">📸</div>
   Replace with: <img src="kanika1.jpg" alt="caption">
   inside the polaroid-img div.


HOW TO ADD MUSIC
----------------
1. Put a .mp3 file in this folder (e.g. song.mp3)
   Recommended songs: "Tum Hi Ho", "Tera Ban Jaunga",
   "Kesariya", "Raataan Lambiyaan" — or any song you both love.

2. Open index.html in a text editor.

3. Find this line:
   <source src="song.mp3" type="audio/mpeg">

4. Change song.mp3 to your filename.

5. Save and refresh. The ♪ button will now play music!


HOW TO WRITE YOUR OWN LETTER
-----------------------------
1. Open index.html in a text editor.
2. Find the comment that says:
   <!-- ✏️ PARTH — REPLACE THIS WITH YOUR OWN LETTER -->
3. Replace the paragraph text below it with your letter.
4. Save and refresh.
